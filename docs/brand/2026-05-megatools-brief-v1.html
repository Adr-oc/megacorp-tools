MegaTools
Brief de diseño · v1 · Mayo 2026
Dirección: Workbench

Portal interno de herramientas para el grupo MEGACORP.

Una app multi-tenant con un registry de herramientas que va creciendo sprint a sprint. Construida sobre Next 16 + Tailwind 4 + shadcn — pero rediseñada con personalidad propia: utility-first denso al estilo Linear, sidebar contextual, ⌘K command palette, y un acento de color que cada usuario elige al entrar y lo acompaña en todo el portal.

Tres ideas que sostienen el diseño

  • 01
    Sidebar contextual, no redundante. El riel izquierdo cambia según el módulo. En home muestra solo el switcher de org y un atajo a lo reciente. Dentro de una app, muestra info y acciones de esa app. En Configuración muestra las secciones (Tu cuenta · Organización). El mismo riel — distinto contenido — nunca duplica lo que ya está en el área principal.
  • 02
    Accent color por usuario. Cada persona elige uno de seis colores al onboardear. El accent tiñe CTAs, estados activos, badges y resaltes en toda la UI. Se guarda con el perfil — viaja con el usuario.
  • 03
    Guía siempre a mano. Onboarding de 3 pasos (bienvenida → elegí tu color → tour guiado de 6 pasos) y un botón ❓ permanente en el topbar que reabre el tour. Nadie debería sentirse perdido en su primer minuto.

Sistema de logo · 3 versiones

El mark del grupo en tres tratamientos. El brand (a color) queda reservado para marketing — landing pública y mails. Adentro del producto siempre va la versión mono, que se adapta al theme manteniendo carácter (alto contraste, sin rastro de los colores originales, meridianos gruesos).

Construcción: 4 cuadrantes + 3 meridianos (círculo, horizontal, vertical). Stroke 2.5 en mono para que la geometría se lea con presencia, 1.5 en brand. Tamaños mínimos: 20px (sidebar), 28px (topbar), 36px (header / login), 80px+ (onboarding / marketing). Siempre rendereado como SVG inline (no PNG, no img), nunca con CSS filters.

Accent · 6 presets

El usuario elige el suyo en el segundo paso del onboarding. Default Mustard. Para botones primarios, el sistema calcula automáticamente el color de texto adecuado (negro o blanco) según la luminancia del accent — los seis presets están balanceados para garantizar contraste AA en ambos modos.

Stack & restricciones

Next 16 (App Router, TypeScript estricto) · Tailwind 4 + tokens OKLCH en globals.css · shadcn/ui sobre Base UI (usar lo que ya existe — no rehacer botones) · Better Auth con organización + magic link (Resend para email) · Drizzle + Postgres · Playwright para E2E · @dnd-kit para drag-drop en PDF · cmdk para la palette · next-themes para light/dark.

Todo el copy es en español rioplatense. Mantener.
